# PC操作模拟器

---

使用pynput来执行用户自定义的一系列鼠标和键盘操作。

---

目前已经实现了单独的鼠标录制功能和键盘录制功能，其中鼠标方面用户自行在屏幕上点击录制，随后系统执行。
键盘录制方面，用户修改配置文件，在配置文件里定义相关操作，随后启动。系统根据配置文件中的内容执行操作。

---

```python
现在开发view.py

界面标题是“PC操作器”，居中，向下是两个按钮，分别是鼠标操作和键盘操作。界面底下是一行声明“@nottky”。

用户点击鼠标操作，进入鼠标操作界面，一个输入框，一个开始录制按钮，一个开始执行按钮。
```
