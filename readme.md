# PC 操作器 (PC Mouse & Keyboard Simulator)

一个基于 PyQt5 的图形化鼠标键盘自动化工具。支持录制屏幕点击位置并自动重复点击，也支持通过 JSON 配置文件执行复杂的键盘按键序列回放，非常适合重复性操作场景。

## 功能特点

- **图形用户界面** – 基于 PyQt5，操作简单直观。
- **鼠标模拟**
  - 录制任意数量的屏幕点击位置。
  - 可设置重复次数和点击间隔。
  - 执行前有 3 秒倒计时，方便切换目标窗口。
- **键盘模拟**
  - 通过 `config.json` 灵活定义按键序列。
  - 支持普通字符、方向键（含自定义缩写）、功能键等。
  - 可设定整个序列的重复次数。
  - 首次运行自动生成配置文件模板，图形界面内嵌详细编写说明。
- **独立模块** – `touch.py` 和 `board.py` 可以脱离 GUI 直接运行或集成到其他脚本中。

## 安装与依赖

1. 克隆或下载本项目：
   ```bash
   git clone <仓库地址>
   cd sometky-pcmouseandboardsim
   ```

2. 安装依赖（推荐使用虚拟环境）：
   ```bash
   pip install PyQt5 pynput
   ```
   > 需要 Python 3.6 及以上版本。

## 快速开始

### 启动图形界面

```bash
python view.py
```

主窗口包含两个入口：“模拟鼠标”和“模拟键盘”。

### 鼠标模拟

1. 点击主界面的 **模拟鼠标** 按钮。
2. 在“录制点个数”输入框中填入要录制的次数（例如 `3`），点击 **开始录制**。
3. 根据提示，依次在需要点击的屏幕位置**点击鼠标左键**。每点击一次，终端/控制台会输出记录的位置。
4. 录制完成后会弹出提示。
5. 设置“重复次数”（如 `10`）和“延迟(秒)”（如 `0.5`），点击 **开始执行**。
6. 程序会倒计时 3 秒，请在此期间将焦点切换到目标窗口，之后会自动依次点击录制的位置。

> 注意：录制过程中程序会阻塞等待鼠标点击，请确保点击足够的次数。

### 键盘模拟

1. 准备好 `config.json` 配置文件（若不存在，首次运行键盘模拟时会自动生成模板）。
2. 点击主界面的 **模拟键盘** 按钮，查看页面内的配置文件编写说明。
3. 点击 **开始执行**，程序会读取 `config.json` 并按照配置的序列和重复次数回放按键。
4. 执行前同样有 3 秒倒计时供你切换窗口。

## 配置文件说明 (`config.json`)

`config.json` 的结构如下：

```json
{
    "key_count": 12,
    "key_sequence": ["enter", "la", "ld", "ld", "del", "1", "enter", "lw"],
    "repeat_count": 80
}
```

- **key_count**：按键个数（整数），建议与 `key_sequence` 列表长度一致。
- **key_sequence**：需要依次按下的按键列表，从左到右顺序执行。
- **repeat_count**：整个序列重复执行的次数。

### 支持的按键写法（不区分大小写）

| 类型         | 写法示例                                         |
|--------------|--------------------------------------------------|
| 普通字符     | `"a"`, `"1"`, `"!"`, `"B"`                       |
| 方向键缩写   | `"la"` (左), `"lw"` (上), `"ls"` (下), `"ld"` (右) |
| 功能键名称   | `"enter"`, `"space"`, `"tab"`, `"esc"`, `"shift"`, `"ctrl"`, `"alt"` |
| 其他方向键   | `"up"`, `"down"`, `"left"`, `"right"`            |
| 退格键       | `"del"` 或 `"backspace"`                         |

> 说明：按键序列中每个键会被按下后立即释放，键间延时 0.05 秒，序列重复间隔 0.2 秒。

## 注意事项

- 录制鼠标位置时，当前终端/控制台会提示“请点击第 x/y 个位置”，请根据提示完成点击，否则程序会一直等待。
- 鼠标点击执行与键盘回放开始前的 3 秒倒计时请务必切换到目标窗口，避免误操作。
- 如果 `config.json` 缺失或格式错误，键盘模拟会给出相应提示并自动生成默认模板。
- 强制停止：鼠标自动点击器内置了 `KeyboardInterrupt` 处理，在终端中按下 `Ctrl+C` 可以尝试中止（图形界面中建议关闭窗口或等待完成）。
- 本工具依赖鼠标和键盘的底层模拟，部分系统可能需要授予辅助功能权限（如 macOS 的辅助功能权限）。
